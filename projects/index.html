Projects

   Timerdo - I've built and maintain a CLI application, "A minimalist to-do list with a built-in timer" that I use daily to track my routine. It's a Python package easily installed with `pip install timerdo`. It's built in Python and relies on a SQLite database to keep all records. It's strongly based on SQLMoldel and Typer.
   Top Drivers of Bike Sharing Demand in Seoul - I built a regression model using Machine Learning techniques to understand the key drivers of bike sharing demand in Seoul. The prototype was made in Python with Ipynb, using Pandas, Numpy, Scikit-Learn, Matplotlib and Seaborn.
   Insights on Financial Complaints - I performed a simple Exploratory Data Analysis. I was looking to provide insights for an imaginary company that handles customer complaints and offers its services to companies in the financial sector. Technically speaking, I used the Google BigQuery API to consume the data and deployed a simple interactive report with Streamlit and Heroku.
   Analysing Supermarket Sales - I performed a simple Exploratory Data Analysis to understand the difference in sales between three branches of the same supermarket. I raised three hypothesis and performed a non-parametric Mann-Whitney test to confirm my main hypothesis - as I was working on a non-normal distribution. I used Python with Pandas, Numpy, Matplolib and seaborn and produced a report with my conclusions.
